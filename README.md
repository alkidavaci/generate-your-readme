  ## Description
  üîç This application is a node.js that uses user input from inquirer to create a README.md file for user repository. User is asked to enter the title of project,Description, Installation, Usage, License, Contributing, Tests and Questions. If user doesn't want to add Installation, Usage Tests or Questions the application will create a README file with other inputs. There is a validation of the email.
